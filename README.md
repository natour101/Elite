# Elite Luxury Store

مشروع Flutter احترافي لمتجر فاخر للمنتجات الأصلية فقط، مع:

- واجهة متجر `Mobile First` للويب
- لوحة تحكم إدارة لويندوز
- بيانات وهمية جاهزة للتجربة
- طلب المنتج عبر واتساب
- بنية قابلة للتوسعة نحو Firebase لاحقًا

## التقنيات المعتمدة

- `Flutter`
- `Riverpod`
- `GoRouter`
- `Google Fonts`
- `Url Launcher`

## لماذا Riverpod؟

تم اختيار `Riverpod` لأنه مناسب لمشروع متعدد المنصات، سهل الاختبار، ويوفر فصلًا واضحًا بين طبقة العرض والبيانات بدون تعقيد زائد.

## التشغيل

1. ثبّت Flutter SDK وأضفه إلى `PATH`.
2. من داخل المشروع شغّل:

```bash
flutter pub get
flutter create . --platforms=web,windows
```

هذا الأمر مهم في هذه البيئة لأن Flutter SDK غير متاح حاليًا، لذلك لم أتمكن من توليد ملفات المنصات آليًا من هنا.

3. للتشغيل على الويب:

```bash
flutter run -d chrome
```

4. للتشغيل على ويندوز:

```bash
flutter run -d windows
```

5. لبناء نسخة EXE:

```bash
flutter build windows --release
```
